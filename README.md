## Prerequisites

- git
- yarn

## Getting started

- Clone this repository
  ```sh
  git clone https://github.com/nonceblox/artefy-frontend.git
  ```
- Navigate to `artefy-frontend`
  ```sh
  cd artefy-frontend
  ```
- Install dependencies
  ```sh
  yarn
  ```
- Copy `.env.example` to `.env` and fill in the values

## Run

```sh
yarn start
```
